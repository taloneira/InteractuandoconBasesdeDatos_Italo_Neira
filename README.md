# InteractuandoConBasesDeDatos

   Para la versión PHP del proyecto:

1) Inicialice su base de datos importando el documento initialize.sql en PHPMyAdmin

2) Crea tus usuarios abriendo la página create_users.php en un navegador web

Para la NodeJSVersión del proyecto:

1) Ejecute el siguiente comando en su terminal:
    npm instalar

2) Ejecute el siguiente comando en una ventana de terminal separada para abrir una nueva conexión MongoDB (y dejarla abierta):
    mongod (o sudo mongod)
   
3) Ejecute el siguiente comando desde la carpeta raíz del proyecto:
    nodo servidor / createUsers.js
   
4) Luego, ejecute este otro comando en esa misma ventana de terminal:
    nodo servidor app.js
   
    - Después de hacer todo esto, ¡ya estás listo!
